# 🕒 DailyKnowly - 総合学習支援 Web アプリケーション

[![Java](https://img.shields.io/badge/Java-17-orange.svg)](https://www.oracle.com/java/)
[![Jakarta EE](https://img.shields.io/badge/Jakarta%20EE-10-blue.svg)](https://jakarta.ee/)
[![MySQL](https://img.shields.io/badge/MySQL-8.0-blue.svg)](https://www.mysql.com/)
[![Render](https://img.shields.io/badge/Deployment-Render-brightgreen.svg)](https://render.com/)

DailyKnowlyは、Java Servlet (Jakarta EE) とJSP、そしてクラウドデータベース（Aiven MySQL）を組み合わせた、Webベースの総合学習支援アプリケーションです。  
効率的な暗記をサポートするフラッシュカード機能やスケジュール管理機能に加え、Fetch APIを用いた完全非同期通信による「集中タイマー記録システム」などを搭載しています。

---

## 🌐 実機デプロイ環境 (Live Demo)

本アプリケーションは、RenderおよびAiven MySQLを用いて実際にインターネット上に公開しています。

👉 **[DailyKnowly 実機Webサイトはこちら](https://dailyknowly.onrender.com/)**  
※無料プランのサーバーを使用しているため、最初のページ読み込み（インスタンスの起動）に **1〜2分** ほど時間がかかる場合があります。

### 🔑 レビュー用テストアカウント
アプリの挙動をスムーズにご確認いただくためのテスト用共通アカウントです。
- **ユーザーID（メールアドレス）:** `test`
- **パスワード:** `password`

### ⚠️ 【重要】データベースが接続エラーになる場合の注意点
本アプリのデータベースには **Aiven MySQL（無料プラン）** を採用しています。Aivenの仕様により、**一定期間（数日間）アクセスがない場合、DBサーバーが自動的に「POWER OFF（一時停止）」状態に移行する**仕組みになっています。

もし画面上でエラーが発生したり、データの保存・読み込み（タイマーや問題一覧など）が失敗する場合は、DBサーバーがスリープしている可能性が高いです。

その場合は、大変お手数ですが **GitHubのIssue、または直接メッセージ等で開発者までご連絡いただけますと幸いです。** 開発者側で速やかに管理画面から「Resume（再開）」を行い、サーバーを復帰（`RUNNING` 状態に）いたします。

## 🚀 実装機能一覧

### 1. 🧠 学習開始（アクティブリコール）
- **フラッシュカード学習システム**
  - 登録した単語カードを順に表示し、クリックで解答を表示する「Anki」風の学習システムを搭載。
  - 「覚えた」「忘れた」をユーザー自身が判定し、効率的に次の問題へ進むことができます。
- **科目限定・復習モード**
  - 全問題からではなく、選択した科目ごとの復習対象カードのみを抽出して効率的に学習可能です。
- **模擬試験モード**
  - 科目内の全問題をランダムにシャッフルして出題する試験形式を導入。
  - 「答えを見る」→「正解・不正解を判定」という自己判定プロセスにより、表記揺れの影響を受けないストレスフリーな反復練習を実現しました。
  - 終了後には正解率を表示し、ワンクリックで即座に再挑戦が可能です。

### 2. 📝 カード作成・科目管理
- **教材のパーソナライズ**
  - 学習したい「科目」を自由に新規追加。
  - 追加した科目ごとに、フラッシュカードの問題と答えをいつでも作成・ストックできます。

### 3. 🔍 問題一覧（シェア＆アップデート機能）
- **マイ問題と公開問題の切り替え**
  - 自身が作成した「マイ問題」と、他ユーザーが共有した「公開問題」を科目ごとに一覧表示。
- **柔軟な管理機能**
  - マイ問題の編集・削除に加え「問題の一般公開」が可能。
  - 公開問題のワンクリック・インポート機能、科目自体の公開範囲設定に対応。

### 4. 📅 カレンダー ＆ 今日の予定
- **スケジュール管理**
  - アプリケーション内で独自の「カレンダー」を生成し、予定のCRUD（作成・編集・削除）にフル対応。
  - ダッシュボード（ホーム画面）には「今日の予定」が自動抽出されて一覧表示されます。

### 5. ⏱️ 集中タイマー ＆ 学習記録（グラフ可視化）
- **ポモドーロ・タイマー機能**
  - 学習時間と休憩時間をユーザー好みにカスタム設定可能。自動でモードが切り替わります。
- **Fetch APIによる完全非同期の自動保存**
  - 画面をリロードさせることなく、裏側で静かにJava Servletへ学習データをPOST送信します。
- **学習データの自動蓄積と可視化**
  - 学習開始およびタイマーで計測した時間を自動でデータベースへ登録。累積学習記録をグラフを用いて視覚的に振り返ることができます。

### 6. 学習モードの拡充（復習・模擬試験）
- **科目限定・復習モード**
  - 全問題からではなく、選択した科目ごとの復習対象カードのみを抽出して効率的に学習可能です。
- **模擬試験モード**
  - 科目内の全問題をランダムにシャッフルして出題する試験形式を導入。
  - 「答えを見る」→「正解・不正解を判定」という自己判定プロセスにより、表記揺れの影響を受けないストレスフリーな反復練習を実現しました。
  - 終了後には正解率を表示し、ワンクリックで即座に再挑戦が可能です。
